# 📊 Paisabazaar Banking Fraud Detection & Risk Analysis

## 🎯 Objective

This project focuses on analyzing customer financial data to identify credit risk patterns, detect potential fraud, and support better decision-making in banking systems.

Analyze customer financial data to identify credit risk and detect fraud patterns.

## 🛠️ Tools Used

* Python
* Pandas
* NumPy
* Seaborn
* Matplotlib

---

## 📌 Project Steps

1. Data Cleaning & Preprocessing
2. Variable Understanding
3. Data Wrangling
4. KPI Analysis
5. Data Visualization

---

## 📊 Key Insights

* High debt and delayed payments → poor credit score
* Payment behavior strongly impacts creditworthiness
* Credit utilization is a major risk indicator

---

## 🚀 Outcome

* Identified high-risk customers
* Improved understanding of credit behavior
* Useful for banking decision-making
